Routine Track 隐私政策

最后更新日期：2024年6月4日

1. 信息收集

Routine Track 尊重您的隐私。本应用：

  • 不收集任何个人信息
  • 不收集任何设备信息
  • 不收集任何使用数据
  • 所有数据仅存储在您的设备本地

2. 数据存储

本应用的所有数据（包括习惯记录、设置等）都存储在您的设备本地，不会上传到任何服务器。

3. 数据安全

由于所有数据都存储在本地，我们无法访问您的任何信息。请确保妥善保管您的设备。

4. 第三方服务

本应用不包含任何第三方服务或分析工具。

5. 儿童隐私

本应用不针对13岁以下儿童。我们不收集任何儿童的个人信息。

6. 权限使用

本应用仅使用必要的系统权限：

  • 通知权限：用于发送习惯提醒
  • 存储权限：用于保存应用数据

7. 隐私政策更新

我们可能会更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新日期"。
